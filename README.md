### 项目2.2--初级爬虫（图片）(2019-07)
- 说明：
- 1.爬取图片网站为[堆糖网](https://www.duitang.com/)
- 2.理解爬虫逻辑：获取网页源码-->解析-->筛选-->保存
- 3.图片网站为动态加载，不同于小说网站(静态加载)
- 4.程序中url的获取方式：在搜索框输入关键字-->按F12-->点击network-->下滑网页-->左下角出现图片-->右侧的Request URL即为所需url
